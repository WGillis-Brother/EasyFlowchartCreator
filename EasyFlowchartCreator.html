Quick Tutorial

1) Set up the chart
Enter a title and pick Landscape or Portrait.

2) Add shapes
Use sidebar shape buttons to add Square/Rectangle, Diamond, and Circle/Pill nodes.

3) Edit text and properties
Click inside a node to edit text inline. Use Node Properties for text, type, and size. Right-click node → Properties for detailed edits.

4) Move and resize
Drag nodes to move. Drag bottom-right chevron handles to resize.

5) Connect and disconnect
Drag the green connector dot from one node to another to create a wire. Click the red dot on a wire to remove that connection.

6) Navigate the workspace
Click and drag empty workspace to pan. Use mouse wheel over workspace to zoom in/out.

7) Control zoom
Use the sidebar Zoom slider. Use Fit to Screen anytime to refit manually.

8) Save and export
Save Project / Load Project preserve your editable JSON. Export PDF creates a light-mode printable copy.

Short examples
